## Resource management guide

* fairy
* fairy/battle
* pic
* icon/equip
* icon/skillicon
    인게임 리소스의 이름을 변형 없이 그대로 사용합니다.
* image/typeicons
    인형 종류/등급별 이미지. `_r`은 속이 투명합니다.
* spine
    캐릭터-스킨별 폴더 안에 `.png`, `.atlas`, `.skel` 파일 존재.
